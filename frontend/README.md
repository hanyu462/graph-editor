# Frontend

This directory contains the React + Vite application for `graph-editor`.

## Install

```bash
npm install
```

## Commands

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Dependencies

Do not commit `node_modules`.
Install dependencies locally with `npm install` after cloning the repository.

For overall project usage, see the repository root `README.md`.
