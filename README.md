# graph-editor

Graph data and CSV data used with a React + Vite frontend editor.

## Repository structure

- `frontend/`: React + Vite application
- `graphs/`: graph JSON files
- `raws/`: raw CSV data files
- `backend/`: reserved backend directory

## Requirements

- Node.js 20 or newer recommended
- npm

## Getting started

Clone the repository and install frontend dependencies:

```bash
git clone https://github.com/hanyu462/graph-editor.git
cd graph-editor/frontend
npm install
```

## Run the app

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Dependency notes

This repository does not commit `node_modules`.
Dependencies are defined in `frontend/package.json` and locked in `frontend/package-lock.json`.
After cloning, each user should run `npm install` inside `frontend/`.

## Notes

- `frontend/dist/` is generated by the build and is not tracked in Git.
- Environment or secret files such as `.env` are ignored by Git.
